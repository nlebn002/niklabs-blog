# Project

Mission: build and maintain `niklabs-blog`, a personal blog platform with a .NET modular backend and React frontend.

Current scope:
- Blog module: posts, drafts, slugs, tags, and media-backed content.
- Auth module: authentication, authorization, and admin access.
- Frontend: editorial/admin workflows plus public reading experience.
