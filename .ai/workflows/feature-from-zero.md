# Feature From Zero

1. Create `.ai/specs/YYYY-MM-DD-<slug>.md` from `_template.md`, omitting literal angle brackets on Windows.
2. Fill in problem, scope, acceptance criteria, and expected files.
3. Read relevant `.ai/context/` files and nearby code.
4. Add a plan section with ordered implementation steps.
5. Implement backend, frontend, and contract changes in that order unless the feature suggests otherwise.
6. Run focused tests and builds.
7. Review the diff with `.ai/prompts/review-checklist.md`.
