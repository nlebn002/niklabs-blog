# Plan Implement Review

Standard loop:

1. Spec: write or update a feature brief in `.ai/specs/`.
2. Plan: append a concrete implementation plan to the spec.
3. Implement: make the smallest coherent change that satisfies the spec.
4. Verify: run relevant builds, tests, migrations, and client generation.
5. Review: use `.ai/prompts/review-checklist.md` against the diff.
6. Handoff: summarize changes, verification, and known gaps.
